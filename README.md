### Описание проекта
Telegram бот, может показать погоду в указанном городе, 
а также найти фильмы по названию.
Также бот показывает рейтинг топ-250 лучших фильмов.

### Как пользоваться
* Команда '/start' или '/help': стартовая команда. Бот выдает приветствие, а также показывает основное меню.
* Команда '/history': показывает всю историю запросов к боту
* Кнопка в меню 'Погода': показывает погоду в указанном городе.
* Кнопка в меню 'Поиск фильмов': ищет фильмы по названию
* При нажатии на кнопку 'Поиск фильмов', появляется еще 1 кнопка 'Рейтинг фильмов':
* При вводе нужной страницы (от 1 до 13 по 100 на 1 странице) показывает список из 100 лучших фильмов.

   в личном кабинете скопировать свой token в файл .env напротив movie_key

### Как запустить

#### Находясь в корневом каталоге проекта:

- Установить зависимости командой:

      pip install -r requirements.txt

- Запустить бота:

      ввести в терминале: python3 main.py
      либо в модуле main.py нажать: Run 'main'
