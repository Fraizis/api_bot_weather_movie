### Описание проекта
Бот - помощник для Telegram, может показать погоду в указанном городе, а также найти фильмы по названию (название фильма вводится на английском языке).
Также бот показывает рейтинг топ-1000 лучших фильмов во версии IMDB.

### Как пользоваться
* Команда '/start' или '/help': стартовая команда. Бот выдает приветствие, а также показывает основное меню.
* Команда '/history': показывает всю историю запросов к боту
* Кнопка в меню 'Погода': показывает погоду в указанном городе.
* Кнопка в меню 'Поиск фильмов': ищет фильмы по названию(англ.)
* При нажатии на кнопку 'Поиск фильмов', появляется еще 1 кнопка 'Рейтинг фильмов':
* При вводе нужной страницы (от 1 до 10 по 100 на 1 странице) показывает список из 100 лучших фильмов.

### Для работы бота были взяты следующие API:
Чтобы пользоваться всеми возможностями бота, вам нужно получить API ключи из следующих источников:


### Поиск погоды:

url_погоды = https://openweathermap.org/

example = https://imgur.com/Cxp4z0I

### Поиск фильмов:

url = https://rapidapi.com/ntd119/api/imdb-com

example = https://imgur.com/ZGW0aXt

### Список фильмов по рейтингу:

url = https://rapidapi.com/robotfa-robotfa-default/api/imdb-top-1000-movies-series

example = https://imgur.com/V71Bwg5


### Как запустить

Запуск бота производится в файле main.py с помощью команды: Run 'main'

* Для запуска и работы бота вам нужно установить в терминале:

* pip install pytelegrambotapi
* pip install peewee
* pip install python-dotenv

* И установить библиотеку:
* install requests
