## Telegram bot для поиска фильмов и погоды

### Описание проекта

Бот для Telegram, может показать погоду в указанном городе, 
а также найти фильмы по названию. Также бот показывает 
рейтинг топ-250 лучших фильмов.

### Как пользоваться

* Команда '/start' или '/help': стартовая команда. Бот выдает приветствие, 
а также показывает основное меню.
* Команда '/history': показывает всю историю запросов к боту
* Кнопка в меню 'Погода': показывает погоду в указанном городе.
* Кнопка в меню 'Поиск фильмов': ищет фильмы по названию
* При нажатии на кнопку 'Поиск фильмов', появляется еще 1 кнопка 'Рейтинг фильмов':
* При вводе нужной страницы (от 1 до 13 по 20 фильмов на 1 странице) показывает список из 250 лучших фильмов.

### Запуск и настройка бота: 

1. Переименовать файл env.template в .env 
2. Зарегистрироваться на https://openweathermap.org/
3. Зарегистрироваться на https://kinopoiskapiunofficial.tech/ , 
   в личном кабинете скопировать свой token для пользования API
   в файл .env напротив movie_key 
4. Находясь в корневом каталоге проекта, установить зависимости командой:

       pip install -r requirements.txt

5. Запустить бота:

       ввести в терминале: python3 main.py
       либо в модуле main.py нажать: Run 'main'
